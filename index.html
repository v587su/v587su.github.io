Selected Projects

Deep Learning based Semantic Code Search

This project proposed a neural model for semantic code search. When the semantics of code (e.g., identifier names) are ambiguous, previous works need to relearn the structural information from lexical tokens of code, which is extremely difficult for a model without any domain knowledge. In contrast, Our model encodes both the semantics and structures of code, by representing code with AST paths. Our model significantly outperforms both DeepCS, the first approach that applies deep learning to code search task, and CARLCS, a state-of-the-art approach that bases on deep learning. The paper is currently under submission. [PDF]


Requirement based Library Recommendation

This project identified a new perspective on library recommendation: recommending useful libraries given requirement descriptions of the software project. It helps to avoid some key problems in existing models including requirement insufficiently matching and cold-start problem. With deep learning (DL), it is practical for models to understand the requirements of projects. We proposed a DL model for library recommendation based on requirement descriptions. The experiment results show that the model can recommend libraries accurately with a SuccessRate@10 of 90.4%, which proves that this perspective has practical values. [PDF]





Library Search using Dependency Graph

This project studies how to improve the library search with the dependency graph of libraries. Library search is important to developers, especially starters. The search results on Google are usually too noisy with unrelated texts. To identify the suitable libraries directly, we proposed a neural IR model with the dependency graph. The Graph-enhanced search has been successfully applied in fields like product search. The dependency graph of the library provides rich information on the usage of libraries, which helps the model to better understand the functions of the library.





Smart Resource Allocation among Cities in Region

This project focused on resource allocation in a region using smart techniques. The goal of this project is to provide advice to policymakers for the regional development of innovation. We built a machine learning model with XGBoost to predict the innovation ability of cities based on their innovation-related statistics. With the prediction model, we used optimization algorithms like Simulated Annealing to find the optimal solution for resource allocation.
